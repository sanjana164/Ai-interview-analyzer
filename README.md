# 🚀 AI Interview Analyzer

AI-powered interview preparation platform that analyzes resumes and job descriptions to generate personalized interview questions, identify skill gaps, and create preparation plans.

## ✨ Features

* Resume PDF upload
* AI-generated interview questions
* Match score analysis
* Skill gap identification
* Personalized preparation plan
* Resume PDF generation
* Secure JWT authentication

## 🛠 Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* Gemini API
* JWT Authentication
* Render

## 🚀 Installation

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
```
